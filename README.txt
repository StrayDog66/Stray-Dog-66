  PASO 1 — Descargar
  ──────────────────
  Bajá esta carpeta completa a tu computadora.
  Guardala en: Documentos/STRAY DOG/

  Download this entire folder. Save it to:
  Documents/STRAY DOG/


  PASO 2 — Sitio web
  ──────────────────
  Abrí 01_website/index.html en cualquier navegador para ver
  el sitio. Para publicarlo online: subir a Vercel, Netlify,
  GitHub Pages (gratis) — solo arrastrar la carpeta.

  Open 01_website/index.html in any browser. To publish:
  drag the folder to Vercel, Netlify, or GitHub Pages.


  PASO 3 — Logos
  ──────────────
  Los SVG de 04_logos-svg/ son vectoriales — escalables al
  infinito. Los podés usar en Illustrator, Figma, Canva, o
  cualquier programa de diseño. Para imprenta, exportar a PDF.

  SVG files in 04_logos-svg/ are vectors — infinite scaling.
  Use them in Illustrator, Figma, Canva. For print, export
  to PDF/EPS.
